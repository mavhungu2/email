# email-api 
A mock email API    

## Steps
This sample email api.    

### 1) Clone the repository, install node packages  and verify routes locally

``` 
//on local
git clone https://github.com/zowe/sample-node-api
cd email
npm install
npm start
```

Refer to api-sample-calls.json for testing API calls
